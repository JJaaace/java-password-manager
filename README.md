# Java Password Manager

A simple command-line password manager built in Java that allows users to store and view account credentials.

## Features
- Add account names and passwords
- View saved credentials
- Menu-based interface for easy interaction
- Uses ArrayList to store data dynamically

## How It Works
The program runs in a loop and displays a menu with options:
1. Add a password
2. View saved passwords
3. Exit the program

User input is handled using Scanner, and data is stored in ArrayLists.

## Technologies Used
- Java
- ArrayList
- Scanner

## Example Usage
=== Password Manager ===
1. Add Password
2. View Passwords
3. Exit

Enter account name: Gmail
Enter password: mypassword123

Saved Passwords:
Gmail -> mypassword123

## How to Run
1. Compile the program:

javac PasswordManager.java

2. Run the program:
   
java PasswordManager
